# ARCHIVER
Simple script to archive files from a of a specified age from a source to destination while maintaining folder struture.

Writes log to destination folder.

Warning: deletes empty folders!

## Getting Started

Run the with arguments as follows:

python archiver.py <SOURCE_PATH> <DESTINATION_PATH> <AGE_IN_DAYS> <IGNORE_STRING1> <IGNORE_STRING2> <IGNORE_STRING3>

### Prerequisites

What things you need to install the software and how to install them

Requires pandas: 

```
pip install pandas
```
